## プロジェクトについて

体温管理ができるLine Botにて使用している体温を表示するためのサイト。

## 特徴
LINE Botの制御を行っているスプレッドシートから一時的なURLを受け取って正しいURLを受け取った場合に情報を表示するシステムになっています。
